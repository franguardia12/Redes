# Redes
Trabajos realizados en la materia redes (introducción a los sistemas distribuidos). Cátedra Álvarez-Hamelin 2C2025.

# Trabajos

Integrantes: Franco Guardia (@franguardia12), Nicolas Chaia (@NicolasChaia), Ezequiel Aragón (@earagon04), Gonzalo Crudo (@gonzacrudo28), Martín Castro (@Martiin17)

-TP1: implementación de un protocolo de capa de aplicación que permita la transferencia de archivos binarios, que utilice UDP como protocolo de capa de transporte y que contemple mecanismos de entrega confiable de paquetes utilizando las estrategias Stop & Wait y Go-Back-N para la transferencia
-TP2: diseño e implementación de un firewall basado en Software-Defined Networking (SDN) utilizando el controlador POX y el emulador de red Mininet para la generación de la topología en la que hacer las pruebas. El firewall permite filtrar tráfico de red mediante reglas configurables en un archivo JSON, y utiliza el protocolo OpenFlow para instalar reglas de flujo en los switches
